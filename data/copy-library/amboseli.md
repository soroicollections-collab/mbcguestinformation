## Key Brand Motifs

- **Kilimanjaro** — the dominant visual and conceptual hero. Africa's highest peak visible from the camp. Every design reference should treat it as an anchor, not a postcard.
- **Maasai cultural references** — the unit names, the landscape, the community. Specific and respectful. Not decorative.
- **Forested setting** — the camp is set in trees. This is unusual for Amboseli (often associated with open plains). Use it as a differentiator.
